# 📗 Step 2 — Getting Started: Courses ka Road Map

## 1. Learning Pedagogy — Yeh Tareeqa Kyun Kaam Karta Hai

Yeh roadmap traditional learning approach se mukhtalif hai. Focus perfection par nahin, balki rapid execution aur practical results par hai.

### Three-Step Learning Approach

#### 📌 Step 1 — Kaam Ka Overview Lein

Pehle poore topic ko high level par samjhein:

* Yeh kya hai?
* Kis problem ko solve karta hai?
* Kahan use hota hai?

#### 📌 Step 2 — Critical Parts Identify Karein

Har cheez equally important nahin hoti.

Identify karein:

* Core concepts
* High-impact skills
* Frequently used workflows

#### 📌 Step 3 — Critical 80% Seekhein Aur Kaam Shuru Karein

* Essential concepts cover karein
* Real projects par apply karein
* Remaining knowledge practical use se acquire karein

> [!IMPORTANT]
> Har topic ki complete mastery mein mahine lag sakte hain.
>
> Effective pedagogy yeh hai:
>
> **Critical 80% seekho → Apply karo → Remaining 20% real work se seekho.**

## 2. Mental Model — AI Era Mein Kaam Ki Teen Layers

AI era mein work ko teen layers mein samjha ja sakta hai.

| Layer   | Kya Hai                                               | Kaise                                   |
| ------- | ----------------------------------------------------- | --------------------------------------- |
| Layer 1 | General Agents se problems solve karwana              | Apna kaam tez karna                     |
| Layer 2 | Specialized AI Workers banana                         | AI ko aap ke liye kaam karwana          |
| Layer 3 | AI Workers ko combine kar ke AI-Native Company banana | Enterprise scale operation create karna |

### Visual Progression

```text
Layer 1
General Agents
      ↓
Layer 2
AI Workers
      ↓
Layer 3
AI-Native Company
```

> [!TIP]
> Har next layer pichli layer par build hoti hai.

## 3. Do Modes Choose Karna

Learning journey shuru karne se pehle yeh decide karna zaroori hai ke aap kis mode mein kaam karna chahte hain.

| Category     | Mode 1 — Problem Solving                | Mode 2 — Manufacturing                   |
| ------------ | --------------------------------------- | ---------------------------------------- |
| Purpose      | AI se apna kaam fast karna              | AI Workers banana                        |
| Audience     | Knowledge Workers + Engineers           | Engineers ya Knowledge Worker + Engineer |
| Primary Tool | Claude Code, OpenCode, Cowork, OpenWork | Claude Code, OpenCode                    |
| Output       | Completed Work                          | AI Worker jo khud work produce kare      |
| Focus        | Productivity                            | Workforce Creation                       |

### Mode 1

Use karein agar:

* Productivity improve karni ho
* Research fast karni ho
* Daily tasks automate karne hon
* Writing, analysis ya coding speed up karni ho

### Mode 2

Use karein agar:

* AI Workers build karne hain
* Workforce create karni hai
* Agent Factory concepts apply karne hain
* AI-Native systems design karne hain

> [!IMPORTANT]
> Mode 2 mein engineering tools hamesha required hote hain.

## Mode 2 Learning Arc

Mode 2 ka complete journey kuch is tarah evolve hota hai:

```text
Build Agent
      ↓
Promote to Employee
      ↓
Connect Nervous System
      ↓
Add Management Layer
      ↓
Enable Dynamic Hiring
      ↓
Validate with Evals
```

### Evolution Stages

#### Stage 1 — Build Agent

Basic agent create karein.

#### Stage 2 — Promote to Employee

Agent ko role, responsibility aur ownership dein.

#### Stage 3 — Connect Nervous System

Events, workflows aur durability add karein.

#### Stage 4 — Add Management Layer

Governance aur coordination introduce karein.

#### Stage 5 — Dynamic Hiring

Workforce ko scalable aur expandable banayein.

#### Stage 6 — Evals

Performance ko objectively measure aur prove karein.

> [!IMPORTANT]
> Agent banana beginning hai.
>
> Goal ek reliable Digital FTE aur eventually AI Workforce build karna hai.

## Key Takeaways

* Learning ka objective complete mastery nahin, rapid execution hai.
* AI work ko 3 layers mein samjha ja sakta hai.
* Mode 1 productivity ke liye hai.
* Mode 2 workforce creation ke liye hai.
* Har learner ke liye alag starting path hai.
* Fast-track path lagbhag 12 ghanton mein pehla Digital FTE ship karwa sakta hai.
* Agent → Employee → Workforce progression Mode 2 ka core roadmap hai.
